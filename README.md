# React + TypeScript + Vite

## Installation

### Starting project

1. **Clone repository**:

   ```bash
   git clone https://github.com/VadimBolotin/funtech.git
   ```

2. **Install dependencies**:

   ```bash
   npm i
   ```

3. **Start project**:

   ```bash
   npm run dev
   ```

### Other Scripts

1. **run ESlint**:

   ```bash
   npm run lint
   ```

2. **run Preview**:

   ```bash
   npm run preview
   ```
